# 🍕 Pizza Sales Analysis using SQL

A SQL-based data analysis project that explores pizza sales data to uncover business insights. The dataset was imported from Excel and organized into four tables: **Orders**, **Order_Details**, **Pizzas**, and **Pizza_Types**.

## 🔹 Key Highlights
- Imported and managed sales data from **Excel**.
- Used **JOINs, GROUP BY, ORDER BY, Aggregate Functions, Subqueries, and Percentage Calculations**.
- Analyzed sales performance, revenue trends, and customer ordering patterns.
- Worked with a **relational database structure** using multiple tables.

## 📊 Insights Generated
- **Top-Selling Pizzas**
- **Revenue Analysis by Category and Size**
- **Average Order Value**
- **Daily and Monthly Sales Trends**
- **Highest Revenue-Generating Pizza Types**
- **Sales Contribution Percentage by Category**

## 🛠️ Tools & Technologies
- **MySQL**
- **Excel**
- **SQL**
- **Data Analysis**

## 🎯 Objective
To improve SQL skills by applying **joins, aggregations, subqueries, and business reporting techniques** on real-world sales data and transforming raw data into actionable insights.
